# 🎯 Number Guessing Game (Java)

A simple console-based Number Guessing Game built using Java.
The program generates a random number, and the user has to guess it within a limited number of attempts.

---

## 🚀 Features

* Random number generation (1–100)
* User input handling
* Feedback system:

  * 📉 Too High
  * 📈 Too Low
  * ✅ Correct Guess
* Limited attempts (7 tries)
* Score tracking system
* Option to play multiple rounds

---

## 🛠️ Technologies Used

* Java
* OOP Concepts
* `java.util.Random`
* `java.util.Scanner`

---

## 📌 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/number-guessing-game.git
   ```

2. Navigate to the folder:

   ```bash
   cd number-guessing-game
   ```

3. Compile the program:

   ```bash
   javac NumberGame.java
   ```

4. Run the program:

   ```bash
   java NumberGame
   ```

---

## 🎮 How to Play

* The system generates a random number between 1 and 100.
* You have 7 attempts to guess the correct number.
* After each guess, you will get a hint:

  * Too High
  * Too Low
* Score is based on the number of attempts taken.
* You can choose to play multiple rounds.
